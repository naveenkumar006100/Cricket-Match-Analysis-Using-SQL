# 🏏 Cricket Match Analysis using SQL

A comprehensive SQL-based data analysis project that explores historical cricket match data to uncover meaningful insights into team performance, player achievements, venue statistics, toss decisions, and scoring trends.

This project demonstrates practical SQL skills by transforming raw cricket match data into actionable business insights using data querying, aggregation, filtering, and analytical techniques.

---

# 📌 Project Objective

The objective of this project is to analyze historical cricket match data using SQL and answer business-oriented questions that help understand:

- Team performance across seasons
- Winning patterns
- Toss impact on match results
- Venue performance
- Player achievements
- Scoring trends
- Match outcomes

The project highlights how SQL can be used as a powerful tool for data analysis and business intelligence.

---

# 📂 Dataset Overview

The dataset contains historical cricket match records where each row represents a completed match.

### Main Attributes

| Column | Description |
|---------|-------------|
| Season | Tournament season |
| Home Team | Home team name |
| Away Team | Away team name |
| Toss Winner | Team winning the toss |
| Toss Decision | Bat or Field |
| First Innings Score | First innings total |
| Second Innings Score | Second innings total |
| Winner | Match winner |
| Player of the Match | Best player |
| Venue | Match venue |
| Start Date | Match date |

# 📈 Key Insights

The SQL analysis generated several meaningful insights:

- Mumbai Indians emerged as the most successful team based on total wins.
- Winning the toss often increased the chances of winning the match.
- Teams preferred choosing to field after winning the toss.
- Wankhede Stadium hosted the highest number of matches.
- AB de Villiers received the highest number of Player of the Match awards.
- Teams chasing targets won more matches than teams defending totals.
- Match outcomes varied significantly across seasons.
- Average first innings scores indicate balanced batting conditions across tournaments.
- Historical data helps identify long-term performance trends.

# 💡 Business Recommendations

Based on the analysis, the following recommendations are suggested:

- Develop venue-specific match strategies.
- Make toss decisions based on historical venue performance.
- Retain consistently high-performing players.
- Improve bowling strategies while defending targets.
- Strengthen batting depth for successful run chases.
- Analyze seasonal performance before planning future tournaments.
- Use historical match data to support tactical decision-making.
